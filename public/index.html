LookBook Database Application

LookBook is an online application which allows users to sell or buy books online. Users after registering themselves as sellers, can sell books that they possess. This application integrates GoogleBooks API, so there is no book that you have which google doesn't (except when you are searching for your personal diary :P ). In simple steps, search the book and sell it. Buyer logs in and searches for the book that he/she wishes to buy, gets a list of sellers and with one click he buys that book. Easy isn't it.So start exploring the site
